# 系统设置

本目录记录 CCM 设置中心的产品边界、配置链路和生产化升级。

## 页面边界

- 通知与渠道：飞书报告通知与任务会话通道。
- 统一大模型：全局 Agent、群聊主 Agent、音乐 Agent 的共享模型连接，以及知识库 Embedding 配置。
- 外观与刷新：当前浏览器的主题、轮询周期和低性能模式。
- 系统与重置：真实版本、进程、凭据保护状态及界面偏好重置。

设置页不承载自动开发演练、子 Agent CLI 探针、任务恢复或知识库文档管理；这些能力归各自业务页面所有。
