# 音乐曲库工具栏布局优化

## 问题

曲库的“全部、收藏、播放队列、歌单”与具体歌单分成两行，歌单目录和具体歌单层级重复，产生较大空白并增加视觉跳跃。

## 调整

- 曲库标题与歌曲数量合并为紧凑标题组。
- 搜索、上传和下载保留在标题行，并统一图标与按钮高度。
- 全部、收藏、播放队列放在统一视图栏左侧。
- 我的歌单、具体歌单和新建按钮接在同一视图栏右侧。
- 删除重复的“歌单一行 + 具体歌单下一行”结构。
- 当前视图使用青色底线与轻背景共同标识。
- 手机端整条视图栏支持横向滚动，不压缩按钮文字。

## 验证

- `npm run build:frontend`
- `CCM_MUSIC_URL=http://127.0.0.1:3080/ npm run test:music-render`
- Playwright 继续覆盖收藏、歌单创建、添加歌曲、排序、重命名、播放全部、下载中心、助手抽屉和移动端控件重叠检查。
