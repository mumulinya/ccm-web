# 我的工作台指挥中心改造

## 目标

将原本偏空、信息分散的“我的工作台”改造成可直接发起目标、处理异常、跟进执行、查看交付和进入常用模块的日常指挥中心。改造只调整前端信息架构与呈现，继续复用现有工作台 API 和任务操作接口。

## 已完成

- 顶部增加“需要处理、正在推进、今日交付、运行资源”状态概览。
- 将目标输入改为明确的执行入口，保留附件、执行位置、执行前计划和二次确认流程。
- 增加全局助手、任务中心、协作群、对话搜索、工具配置快捷入口。
- 主区域按“待处理、执行中、最近交付”组织任务，保留确认、重试、暂停、恢复、取消和归档等既有操作。
- 右侧集中展示项目、协作群和定时任务，直接呈现运行状态、Agent、成员数和下次调度时间。
- 空状态改为紧凑行动提示，不再占据大块无效空间。
- 桌面端采用主任务区加资源侧栏；移动端使用两列指标、两列快捷入口和单列内容，390px 下无横向溢出。
- 使用 Lucide 图标和中性多色状态语义，避免单一紫色氛围与装饰性卡片堆叠。

## 保留行为

- 数据继续来自 `/api/usability/workbench`，15 秒自动刷新保持不变。
- 任务创建仍先生成执行计划，用户确认前不会执行。
- 未修改项目、群聊、定时任务和历史任务数据。
- 本次验证没有提交真实任务，也没有调用付费 Provider。
