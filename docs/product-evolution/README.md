# 产品演进设计文档

该目录保存早期按版本号持续演进的设计文档。它们用于解释架构和产品决策的演变，不作为当前实现是否完成的唯一依据；当前实现与验收记录优先查看 [Main Agent Workchain](../main-agent-workchain/README.md) 及对应源码测试。

| 目录 | 内容 |
| --- | --- |
| `agent-foundations` | Agent 决策、意图、推理、运行内核和会话生命周期 |
| `collaboration` | 协作协议、任务治理、可读性和运行可见性 |
| `global-agent` | 全局 Agent 总控与运行能力对齐 |
| `group-main-agent` | 群聊主 Agent 的计划、Todo、工具循环和子 Agent 协调 |
| `knowledge-and-memory` | RAG、项目记忆和记忆压缩设计 |
| `platform-and-reliability` | 权限、清理、稳定性和日常可用性 |
| `product-features` | 聊天、代码变更、命令中心和宠物功能 |
| `roadmap-and-examples` | Backlog 与端到端示例 |

完整列表见 [CATALOG.md](./CATALOG.md)。
