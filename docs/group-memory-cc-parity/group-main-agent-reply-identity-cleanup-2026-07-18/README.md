# 群聊主 Agent 回复身份头精简

## 结果

- 群聊主 Agent 的普通回复不再重复展示“协”头像、“协调者”和默认回复状态。
- 项目 Agent 回复继续展示 Agent 身份，便于在多人协作消息中区分来源。
- 系统消息继续展示系统身份和状态。
- 本次变更仅调整消息展示，不改变协调者角色、任务分派、会话标题或记忆上下文。

## 原因

群聊会话顶部和当前页面已经确定主 Agent 所属范围。每条主 Agent 回复再次展示固定“协调者”身份没有新增信息，并占用消息首行空间。项目 Agent 的名称仍有来源辨识价值，因此不做统一隐藏。

## 验证

- 前端 production build 通过。
- 本地界面检查到 5 条主 Agent 回复、0 个可见身份头、0 个“协调者”身份文本。
- 浏览器控制台无 error 日志。
