# 全会话可配置头像

## 范围

- 设置中心的“外观与刷新”增加用户头像和 Agent 头像配置。
- 支持六个预设头像和本地 PNG、JPEG、WebP、GIF 图片。
- 本地图片限制为 512 KB，并保存到当前浏览器的界面偏好中。
- 群聊、全局 Agent、项目 Agent 使用同一份实时头像设置。

## 展示规则

- 用户消息展示用户头像。
- 群聊主 Agent、全局 Agent 和项目 Agent 回复展示 Agent 头像。
- 群聊中的项目 Agent 名称继续保留，用于区分具体消息来源。
- 思考占位和非对话内部事件不强制展示头像。

## 数据与边界

- 存储键为 `ccm-chat-identity-v1`。
- 设置只影响当前浏览器界面，不进入 Agent 提示词、会话记忆或压缩上下文。
- 无设置或旧数据无效时回退为 `👤` 和 `🤖`。
- 浏览器存储失败时回滚本次修改，不显示假成功状态。
- “恢复界面默认设置”会同步恢复默认头像。
