# 宠物参考图完整预览修复

## 问题

宠物生成窗口把参考图容器固定为 `150px` 高。较高的照片只能在很窄的区域内查看，人物或宠物主体容易出现在可视区域之外，看起来像图片只显示了一半。

## 修复

- 有图片时取消固定高度，按照图片原始宽高比自适应显示。
- 容器最大高度为 `320px`，移动端为 `280px`；超大图片等比缩小，不裁切。
- 图片使用透明棋盘背景，便于观察透明边缘。
- 增加“查看完整参考图”按钮。
- 完整预览使用独立遮罩层，图片限制在 `94vw × 90vh` 内并保持原始比例。
- 关闭生成窗口时同步关闭完整图片预览。

## 验证

- 使用 `1536x2288` 竖向图片验证预览比例。
- 自动比较渲染宽高比与图片原始宽高比，误差要求小于 `0.02`。
- 验证完整参考图遮罩可以打开和关闭。
- `npm run build:frontend`：通过，2076 个模块。
- `node scripts/pet-workspace-companion-render-regression.mjs`：桌面端、移动端、完整比例和放大预览检查通过。
- 未创建真实宠物任务，模型调用与付费 Provider 调用为 0。
