# 音乐页顶部与弹幕简化

## 问题

- 顶部“正在播放、本地曲库、我的歌单、音乐助手、播放队列”与曲库内部标签重复。
- 点击顶部入口会改变下面曲库标签，形成两组控件控制同一状态的错觉。
- 封面漂浮区域把逐行歌词也作为气泡展示，与歌词面板重复。

## 调整

- 删除顶部重复导航，不再从顶部切换曲库、队列或歌单。
- 顶部改为纯信息栏，只显示当前播放状态、歌曲名和歌手。
- 音源切换、天气和陪伴时间继续保留。
- 曲库、收藏、播放队列和歌单仅由曲库内部唯一一组标签管理。
- 音乐助手仅由右侧抽屉按钮控制。
- 删除歌词到漂浮气泡的写入链路。
- 封面漂浮区域只接收真实弹幕类型，歌词继续在独立歌词面板显示。

## 验证

- `npm run build:frontend`
- `CCM_MUSIC_URL=http://127.0.0.1:3080/ npm run test:music-render`
- 回归明确检查重复顶部导航不存在、当前歌曲信息栏存在，随后继续覆盖曲库、歌单、下载中心、助手抽屉和移动端布局。
