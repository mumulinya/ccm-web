# 工具与扩展文档

该目录统一保存 MCP、Skill、第三方 Agent CLI、权限控制和技能市场相关记录。

| 目录 | 内容 |
| --- | --- |
| `skills-and-prompts` | 内置 Skill、角色 Skill、提示词瘦身和内外部边界 |
| `marketplace-and-installation` | skills.sh、Smithery、来源注册、安装与更新 |
| `authorization-and-audit` | 工具授权、调用范围、审计和验证门禁 |
| `runtime-adapters` | Claude Code、Codex、Cursor 和外部 Runner 适配 |
| `runtime-sync-and-delivery` | MCP 配置下发、运行时快照、重同步和工具目录刷新 |
| `evidence` | 页面与运行验证截图、报告 |

完整列表见 [CATALOG.md](./CATALOG.md)。内置 MCP 在主 Agent 链路中的业务说明位于 [工具配置](../main-agent-workchain/operations-and-integrations/tool-configuration/)。
