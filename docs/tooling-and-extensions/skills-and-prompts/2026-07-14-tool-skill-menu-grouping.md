# 工具与 Skill 菜单分组整理

## 目标

解决工具配置侧栏将 Skill、MCP 和运行治理入口交叉排列的问题，让用户能够按业务类别快速定位入口。

## 调整

- 总览：运行概况。
- 工具与连接：内置核心工具、MCP 连接中心。
- Skill：Skill 管理、外部 Skill 包、技能商城。
- 治理与验收：授权总览、链路验收、调用审计、Agent 运行时。
- 保留全部原有页面标识、统计数据和点击行为，仅调整菜单顺序与视觉分组。
- 移动端隐藏分组标题，保留原有横向滚动入口，避免挤占有限高度。
- 更新工具配置渲染回归，不再依赖固定的运行统计数字，并增加菜单分组、顺序和移动端布局断言。
