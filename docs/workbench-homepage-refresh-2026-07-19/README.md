# 工作台作为 CCM 首页

## 行为

- 直接打开 CCM 根地址时进入“我的工作台”。
- 用户在任意页面执行浏览器刷新后，统一回到“我的工作台”。
- 刷新回首页时 URL 同步为 `?tab=dashboard`，不会再次恢复刷新前的标签。
- 外部直接打开带 `?tab=projects`、`?tab=groups` 等参数的链接时仍尊重深链接，项目、群聊和任务定位能力不受影响。
- 应用内正常切换标签不强制跳回首页。

## 验证

- 前端生产构建通过。
- 首页专项自测通过。
- 浏览器验证普通深链接可以进入指定页面，随后刷新会回到工作台。
- 没有修改任何任务、会话或项目数据。
- 付费模型调用：`0`。
