# CCM 应用图标统一

## 目标

用用户确认的 CCM 字母几何标替换项目中的通用闪电和 Sparkles 品牌占位符，使浏览器、登录页、工作台和桌面窗口使用同一身份。

## 图形结构

- 左右两个 `C` 表示协作边界与双向连接。
- 中央 `M` 表示 CCM 主体与主 Agent 控制核心。
- 深墨色背景保持与登录页和终端工作台一致。
- 薄荷绿、青色和暖白用于区分左右边界与中央核心。
- 矢量源适配 favicon 和页面品牌位，1024px PNG 用于系统窗口与 Apple touch icon。

## 接入范围

- 浏览器 `favicon.svg` 和主题色。
- 首屏加载遮罩。
- 左侧导航品牌入口。
- 登录页品牌入口及明暗主题。
- Electron 宠物窗口图标。
- npm README 品牌图。
