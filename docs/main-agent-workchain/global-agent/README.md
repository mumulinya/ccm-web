# Global Agent

记录全局 Agent 从用户对话到群聊任务监督的业务能力。

- [conversation-and-ui](./conversation-and-ui/)：普通对话、用户可见回复、流式展示与文案。
- [planning-and-status](./planning-and-status/)：计划模式、Todo、状态跟进与进度展示。
- [dispatch-and-supervision](./dispatch-and-supervision/)：向群聊主 Agent 派发、Mission 监督、续跑与目标调整。
- [history-and-memory](./history-and-memory/)：会话历史、结构化完成状态和记忆回写。
- [test-agent-bridge](./test-agent-bridge/)：全局 Agent 对 TestAgent 状态的展示和上层监督桥接。
