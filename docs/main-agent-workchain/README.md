# Main Agent Workchain 长期升级记录

这个目录专门记录“群聊主 Agent / 全局主 Agent 完整工作链路”的长期升级。

目标是让用户在真实生活和真实项目里直接给需求，主 Agent 能稳定完成：

1. 理解需求和影响范围。
2. 形成用户能看懂的计划。
3. 调度工具、群聊或项目 Agent 执行。
4. 检查回执、文件变更、验证和风险。
5. 完成后给用户一份清晰总结。

约定：

- 用户可见区域只放自然、友好的说明。
- Trace、run id、session id、执行器、内部协议和排障字段默认放进“技术详情”。
- 每轮功能升级都在本目录增加一篇稳定记录，写清楚参考来源、实现范围和验证结果。
