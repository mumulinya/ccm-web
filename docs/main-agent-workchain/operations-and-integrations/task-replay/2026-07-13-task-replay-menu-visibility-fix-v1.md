# 任务回放菜单可见性修复 V1

## 问题

任务回放页面、任务参数和回放接口已实现，但主菜单将 `trace-replay` 标记为 `hiddenFromMenu`，用户只能从任务详情间接进入，无法在导航中找到。

## 修复

- 移除任务回放 Tab 的 `hiddenFromMenu` 标记。
- 将任务回放加入菜单管理器的“协作管理”默认分组。
- 保留从任务详情、定时任务运行历史携带任务 ID 直接打开回放的原有链路。

## 验收

- 左侧“协作管理”可直接看到“任务回放”。
- 菜单管理中可管理任务回放的分组与顺序。
- 页面仍支持按任务 ID 或 Trace ID 查询完整回放。
