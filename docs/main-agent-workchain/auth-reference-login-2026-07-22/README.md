# 参考图登录界面实现

Date: 2026-07-22

## 视觉结构

登录页按参考图采用全屏科幻控制台背景、左侧工作区品牌文案和右侧深色本地登录面板。背景使用项目公共资源 `auth-command-center-bg.png`，图片不包含文字、按钮或虚构产品信息。

## 保留的真实功能

- 用户名登录；
- 密码登录；
- 密码显示/隐藏；
- 首次安装注册；
- 管理员开启后的普通账户注册；
- HttpOnly 会话 Cookie 状态提示。

## 明确不加入的参考图功能

参考图中的 SSO 登录、GitHub/Google/Microsoft 登录、记住我、忘记密码、在线统计和外部身份提供商当前没有 CCM 后端能力，因此没有做成可点击的假功能。

## 验证

```text
npm run build:frontend
```

Playwright 桌面视口 `1440x900` 和移动视口 `390x844` 均无页面溢出。登录 Token 仍由服务端 HttpOnly Cookie 保存，不写入 `localStorage`。

登录输入框对项目全局表单样式做了组件内完整重置，并单独处理 Chromium `-webkit-autofill`。普通、聚焦和自动填充状态均保持深色青蓝背景，不会再出现白色输入区域破坏页面配色。
