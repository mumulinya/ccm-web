# Group Main Agent

记录群聊主 Agent 从理解项目需求、创建计划到协调项目 Agent、验收和总结的完整链路。

- [conversation-and-ui](./conversation-and-ui/)：普通回复、消息展示、澄清和友好文案。
- [planning-and-status](./planning-and-status/)：任务接管、Todo、实时状态和进度总结。
- [dispatch-and-coordination](./dispatch-and-coordination/)：项目 Agent 派发、成员问答、结果返回和协调协议。
- [lifecycle-and-recovery](./lifecycle-and-recovery/)：群聊任务生命周期、恢复和同任务续跑。
- [acceptance-and-delivery](./acceptance-and-delivery/)：群聊主 Agent 验收、交付总结和回传质量。
- [test-agent-bridge](./test-agent-bridge/)：群聊主 Agent 向 TestAgent 发起复核并消费结论的链路。
