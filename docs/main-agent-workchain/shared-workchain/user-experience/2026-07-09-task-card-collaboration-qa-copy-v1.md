# 共享任务卡协作问答文案

日期：2026-07-09

## 背景

共享任务卡里有一块执行成员之间的问答摘要，原本标题显示为“Agent 问答”。这属于内部实现叫法，用户更需要知道这是协作过程中的待确认/补充信息。

## 本次调整

- 将共享任务卡的 “Agent 问答” 标题改为“协作问答”。
- 在视觉回归 fixture 中补充一条协作问答摘要，覆盖问题、回答和下一步。
- 增加 Playwright 断言，确认“协作问答”可见，“Agent 问答”不会出现在共享任务卡问答区。
- 增加源码自测守卫，防止后续回退内部术语。

## 验收点

- 群聊和全局任务卡复用该组件时，用户看到的是“协作问答”。
- 技术细节、底层协议和内部命名仍留在技术详情里。
- 任务计划、执行、验收、总结展示链路不受影响。
