# User Visible Agent Role Language v2

本次继续收敛群聊主流程和全局主流程的用户可见文案：用户主视图不再直接暴露“主 Agent / 子 Agent / 群聊主 Agent / 全局主 Agent”等内部角色词，而是统一翻译成“我 / 执行成员 / 协作群 / 项目执行成员”。

## 改动

- 新增 `sanitizeUserFacingPlanStructure()`，让任务卡、状态卡这类结构化展示数据能递归走计划文案清洗。
- 共享任务卡切换到用户主视图结构清洗：
  - “子 Agent 进展摘要”改为“执行成员进展”。
  - “子 Agent 执行计划”改为“执行成员接单计划”。
  - “主 Agent 验收”改为“最终验收”。
- 群聊顶部状态卡改为“协作状态 / 执行成员状态”，当前 Todo、最近动作、下一步会通过同一套文案清洗。
- 群聊流式消息、执行摘要、问答仲裁、协作看板的主视图文案同步改为用户可读说法。
- 全局 Agent 的默认派发、失败、TestAgent 复核兜底文案改为“我会继续跟进 / 我会纳入最终验收”。

## 保留

- 技术详情、协议识别正则和旧运行报告解析仍保留内部词，以便兼容历史数据和排障。
- fixture 中部分旧输入保留，用来验证前端清洗层能把旧数据转换成新展示文案。
