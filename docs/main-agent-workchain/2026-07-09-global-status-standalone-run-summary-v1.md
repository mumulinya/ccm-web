# Global Status Standalone Run Summary v1

## 背景

用户问“现在进展怎么样”时，不一定只有全局开发任务或全局直派任务需要展示。有些全局主 Agent 运行可能正在等待确认、等待补充信息、执行中或监督中，但还没有被任务卡完整覆盖。之前这类运行可能不会出现在状态摘要里，用户会感觉进度断掉。

## 本次修改

- 全局状态追问新增“最近全局运行”段落。
- 只展示未被全局开发任务或直派任务覆盖的运行，避免重复展示。
- 支持展示运行中、监督中、等待确认、等待补充、暂停和失败状态。
- 用户可见文案只展示当前进展和下一步，不展示 `global_run_id`、`supervisor_id`、`trace_id` 或原始协议。
- 普通进展追问仍然不会创建新的 Todo；它只是读取已有状态并回答。

## 验证

- 运行时自测覆盖等待确认的独立全局运行。
- 静态自测覆盖“最近全局运行”桥接。
- 状态摘要继续覆盖内部协议和运行标识不外露。
