# 月薪喵独立 SVG 动作重绘

> 状态：实验方案，已废弃。当前已恢复 `docs/yuexinmiao1-first-version-restore-2026-07-19/README.md` 记录的第一次替换版本。

## 目标

修正补充动作依赖原图集裁帧、动作语义生硬，以及音乐动作佩戴耳机的问题。CCM 业务状态必须呈现真正不同的姿势和场景，同时保留月薪喵的棕色外轮廓、白色中脸与腹部、蓝色眼睛和粗尾巴等角色特征。

## 实现

- 保留来源包的 `spritesheet.webp`，只用于原生待机与基础回退状态。
- 独立绘制 19 个业务状态 SVG，不使用 `<image>`、Base64 WebP、`spritesheet.webp` 引用或图集裁帧。
- 音乐状态改为抬臂、摆尾和身体律动，不显示耳机或音符道具。
- 规划、工作、执行、调试、验收、等待、整理、搬运、提醒、完成、错误和睡眠状态分别使用清单板、电脑、积木与扳手、放大镜与错误目标、检查板、时钟、扫帚、资料箱、铃铛、完成勾、错误面板和被子等独立构图。
- 重新设计共同角色骨架，使补充 SVG 和来源角色使用一致的头身比例、配色与面部特征，而不是通用卡通猫造型。
- 前端、运行时静态目录和 Electron 桌宠资源目录使用同一份生成结果。

## 资源边界

- 来源作者：`kiffin`
- 来源页面：<https://codex-pet.org/zh/pets/yuexinmiao1/>
