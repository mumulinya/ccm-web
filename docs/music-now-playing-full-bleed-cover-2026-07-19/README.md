# 当前歌曲全幅封面

## 目标

当前歌曲封面不再作为卡片中央的小方图展示，而是铺满整个“正在播放”区域，让封面本身承担主要氛围视觉。

## 实现

- 封面使用 `object-fit: cover` 填满播放卡片。
- 播放状态、收藏和队列按钮悬浮在封面顶部。
- 歌曲名、歌手、音源和情绪信息悬浮在底部渐暗区域。
- 使用上下渐暗与轻微左右遮罩确保浅色、深色封面上的文字都可读。
- 长歌曲名保持单行省略，不覆盖相邻内容。
- 播放时仅根据音频强度轻微调整饱和度和亮度，不缩放卡片尺寸。
- 手机与桌面共用全幅封面，移动端标题采用更紧凑字号。

## 验证

- `npm run build:frontend`
- `CCM_MUSIC_URL=http://127.0.0.1:3080/ npm run test:music-render`
- Playwright 检查桌面和手机视口的溢出、控件重叠、歌单、下载中心与音乐助手侧边抽屉。
