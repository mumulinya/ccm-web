# Clause Matrix

Use one row per independently verifiable clause:

| Source | Clause | Owner | Implementation | Verification | Status |
| --- | --- | --- | --- | --- | --- |
| Document and section | Exact rule or contract | Project Agent | File, route, component, or existing behavior | Command, API, browser flow, or review | Implemented, existing, excluded, blocked, or pending |

For API clauses include method, path, authentication, request, response, errors, and state effects. For UI clauses include visibility conditions, interaction, loading, success, empty, error, and responsive states.
